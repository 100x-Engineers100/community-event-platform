**Updated FAQs**

Q: Who can host events on this platform?  
A: Events can be hosted by the 100x team, alumni, and current cohort members. Community-hosted events are reviewed before being published.

Q: Can events on this platform be paid?  
A: Events hosted by the 100x team may be paid. Community-hosted events must be free and are meant for learning and sharing.

Q: How many events can I host per day?  
A: You can submit up to 3 events per day. This limit resets at midnight server time.

Q: Are events reviewed before being published?  
A: Yes. All community-hosted events are reviewed before they are published to the platform.

Q: Can I edit my event after submitting it?  
A: Once an event is submitted, it cannot be edited while it is under review. You can resubmit the event with updates after the review is completed.

Q: What happens if my event is not reviewed in time?  
A: If an event is not reviewed within 7 days, it will expire automatically. You will see the expired status in your dashboard and can resubmit the event with the same or updated details.

Q: When will I see the meeting link for an event?  
A: The meeting link is shared after you successfully register for an event. It is not shown on the public event page.

Q: Can I register for an event that is already full?  
A: No. Once an event reaches its maximum capacity, registrations close and the event is marked as full. You can check back later in case capacity is increased or explore other upcoming events.

Q: Can I host offline or in-person events?  
A: Yes. Events can be online, offline, or hybrid. The event details should clearly mention the format and location.
